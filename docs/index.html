Distributed Metrics &
Log Aggregation System

Real-time ingestion, stream processing, and analytical storage for structured metrics, unstructured logs, Protobuf user events, and arbitrary external API JSON — built with Kafka, Flink, Spark, Hadoop, TimescaleDB, and Spring Boot. Unknown-schema API data flows through Kafka and Flink to HDFS, then a Spark job infers the schema at read time and writes Parquet, S3, or a data warehouse.
